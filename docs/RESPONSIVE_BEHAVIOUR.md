# Responsive behaviour

- **Wide desktop:** three-region helper workspace plus persistent owner rail.
- **Laptop:** task rail collapses; owner review remains visible.
- **Tablet:** workspace and proposal queue become switchable regions.
- **Mobile:** one proposal per page, stacked before/after values, 48px targets, sticky Pause/Stop controls, and no critical horizontal scrolling.
- At 200% zoom the app reflows rather than clipping. Reduced-motion preference disables packet and reveal animation.
