# Demo guide

1. Choose Scholarship, Hospital registration, or College admission.
2. Select an assistance mode and accept the explicit Intent Contract.
3. Copy the synthetic invitation or open the helper workspace.
4. Prepare semantic proposals. Notice that raw values and owner controls are absent.
5. Compare simple/detailed review, privacy, evidence, and consequence views.
6. Approve a safe proposal and see the fictional portal update owner-side.
7. Attempt an owner-only action to see policy blocking.
8. Try Pause and Emergency Stop.
9. Complete the safe proposals and generate a redacted receipt.
