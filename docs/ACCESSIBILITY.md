# Accessibility

TrustMode targets WCAG 2.2 AA: semantic landmarks, ordered headings, visible focus, 44px+ targets, screen-reader labels, live state announcements, text scaling, logical DOM order, non-colour status cues, reduced motion, keyboard-operable dialogs, and readable correction guidance.

Calm Mode removes secondary detail; Larger and Clearer increases type and spacing; high contrast strengthens boundaries. English, Hindi, and Odia cover the core demo journey. Voice input is optional and never permits high-risk approval without visible confirmation.
