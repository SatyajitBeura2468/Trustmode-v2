# Visual system

TrustMode feels like a calm protected path between two people. Two offset frames form the mark; the narrow space between them is the controlled passage through which proposals—not private values—move.

## Tokens

| Role | Value |
| --- | --- |
| Background | `#f7f9fc` |
| Surface | `#ffffff` |
| Ink | `#142033` |
| Secondary | `#607086` |
| Border | `#dce4ee` |
| Trust | `#315ff4` |
| Privacy | `#16785a` |
| Warning | `#9d6200` |
| Danger | `#b4343f` |

Editorial display typography introduces the promise and major decisions; crisp sans-serif UI text carries controls. Layouts use open rails, lists, and fine dividers instead of repeated card grids. Motion is limited to proposal transfer, state confirmation, and focus guidance.

The approved references are in `docs/design-concepts/`: `landing.png`, `workspace.png`, and `mobile-review.png`.
