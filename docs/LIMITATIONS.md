# Current limitations

This release supports controlled fictional workflows and does not claim arbitrary-site compatibility. It is not approved for real sensitive accounts, real documents, OTPs, payments, financial transactions, medical-record changes, or government submissions. It does not replace professional security, privacy, accessibility, legal, or human-factors review and makes no worldwide novelty claim.

The collaboration model is same-browser synchronisation without a production identity or internet transport layer. The extension is restricted to TrustMode controlled portal routes. Moving beyond this boundary requires authenticated multi-device transport, durable encrypted storage, key rotation, abuse controls, observability, incident response, audited portal adapters, and independent assessment.
